Introdução ao Linux: Conceitos e Filosofia

Baseado na aula inaugural com Gustavo Guanabara e Ricardo

Este guia resume os pilares do curso, abordando desde a presença invisível do sistema no nosso cotidiano até a quebra de paradigmas sobre o uso profissional de tecnologia.

1. A Onipresença do Linux

O Linux é o alicerce da computação moderna, presente em:

  • Infraestrutura: 100% dos 500 supercomputadores mais rápidos do mundo rodam Linux.
  • Web: A grande maioria dos servidores que hospedam a internet utilizam o sistema.
  • Dispositivos Móveis: O kernel Linux é a base do sistema Android.
  • Internet das Coisas (IoT): Smart TVs (webOS e Tizen), roteadores e eletrodomésticos inteligentes.
  • Organizações: Google, NASA, IBM e governos utilizam o sistema por sua segurança e código aberto.

2. O Fim da "Guerra" com a Microsoft

A antiga rivalidade foi substituída pela cooperação técnica. A Microsoft é hoje um dos membros da Linux Foundation. O curso enfatiza que o profissional de TI moderno não deve ser "xiita", mas sim versátil, utilizando a ferramenta mais adequada para cada tarefa.

3. Motivos para Aprender Linux

Existem razões estratégicas para dominar este sistema:

  1. Redução de Custos: Substituição de licenças pagas por software livre legalizado.
  2. Hardware Antigo: Capacidade de dar vida nova a computadores com recursos limitados.
  3. Mercado de Trabalho: Alta demanda por profissionais que compreendem ambientes de servidores Linux.
  4. Personalização: Liberdade total para ajustar o sistema às necessidades específicas do usuário.

4. Distribuição de Ensino: Linux Mint

A distribuição escolhida para a prática é o Linux Mint (Cinnamon). O motivo principal é a interface familiar para quem vem do Windows, garantindo que o aprendizado foque no sistema e não na dificuldade de encontrar menus básicos.

5. Software Livre vs. Software Grátis

É importante distinguir que "Livre" refere-se à liberdade do código e não necessariamente à ausência de custo. O ecossistema Linux movimenta bilhões através de serviços, suporte e consultoria especializada.

Analogia dos Veículos

"Não existe um sistema perfeito para tudo. O Linux pode ser o caminhão para grandes cargas (servidores), enquanto o Windows é o carro de passeio. O profissional qualificado sabe dirigir ambos conforme a necessidade da viagem."
